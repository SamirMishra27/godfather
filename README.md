## How to run:

After creating `src/config.ts`:

`yarn install --prod=true` 

`yarn build`

`node .`

## Unit Tests

Install all dev dependencies:

`yarn`

Run tests:

`yarn test`

## Requirements

You will need to have [`node.js`](https://nodejs.org/en/) and [`yarn`](https://yarnpkg.com/) installed to run this
